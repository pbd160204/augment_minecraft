# Augment Survival PvP Datapack

## 버전
- `Minecraft Java Edition 26.2`
- `pack.mcmeta` 데이터팩 범위는 `107.0`

## 관리자 명령
- `/function augment:admin/set_arena`
- `/function augment:admin/reset_arena`
- `/function augment:admin/start`
- `/function augment:admin/pause`
- `/function augment:admin/resume`
- `/function augment:admin/stop`

## 일반 플레이어 사용
- 설명 보기: `/trigger ag_help set 1`
- 일반 증강 메뉴: `/trigger ag_menu set 1`
- 고급 증강 메뉴: `/trigger ag_advmenu set 1`
- `25세 달성` 강화 사용: `/trigger ag_enchant set 1`
- 상점 열기: 대장장이 작업대 우클릭

## 현재 구현된 핵심 기능
- 경기장 설정, 시작, 일시정지, 재개, 중단
- 시작 위치 스폰 저장, PvP 시간 모든 사망 탈락
- 오버월드 외 차원 진입 시 즉시 복귀
- 5분마다 30초 동안 랜덤 중심 월드보더 축소
- 일반 증강 27종, 고급 증강 7종의 클릭형 `tellraw` 선택
- 계열 수치 누적과 주요 계열 보너스
- 상점 시스템
- 주요 제작 금지 레시피 오버라이드

## 이번에 반영된 변경
- `청색맹`
  - 장착 중 슬롯, 양손, 핫바, 일반 인벤토리의 다이아몬드 장비를 네더라이트 장비로 자동 치환
  - 현재 구현은 `아이템 종류` 기준 치환이며, 기존 아이템의 세부 NBT 보존은 보장하지 않음
- `25세 달성`
  - 기존 플레이스홀더 지급 대신 `강화 2회`를 부여
  - `/trigger ag_enchant set 1` 사용 시 왼손과 오른손 아이템에 각각 30레벨 인챈트 테이블에 준하는 랜덤 인챈트 부여
- `투명 망토`
  - 투명화 물약 지속시간을 `5초`에서 `15초`로 변경
  - 상점의 투명화 물약도 동일하게 `15초`로 변경
- `네더 포탈 비활성화`
  - 포탈 블록 삭제 방식 제거
  - 대신 경기 중 네더/엔드 진입 시 저장된 오버월드 위치로 즉시 복귀시키는 방식으로 변경

## 아직 부분 구현인 항목
- `청색맹`
  - 자동 업그레이드는 구현했지만, 기존 인챈트/내구도/커스텀 NBT를 완전 보존하는 형태는 아님
- `25세 달성`
  - 전용 우클릭 아이템 방식이 아니라 `trigger` 기반 사용 방식으로 구현
- `전투 메이지`
  - 실제 가스트 화염구 발사 시스템 대신 `fire_charge` 지급으로 대체
- `30세 달성`
  - 인챈트 테이블 지급까지만 구현
  - `무경험치`, `청금석 32개 고정`, `책장 없이 최대 레벨` 규칙은 아직 미구현
- `투명화`
  - 갑옷까지 완전히 투명하게 만드는 기능은 미구현
